## Short Description

BountyOps shows a practical agent loop for workflows where external state changes quickly. It reads GitHub issues, PRs, CI status, claim markers, inbox-style signals, and local evidence, then decides whether to act, wait, fix, or checkpoint. The demo is static and public-safe: no private Gmail content, tokens, payout credentials, cookies, or bank data.
