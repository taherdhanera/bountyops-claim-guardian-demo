# Security Policy

This repository must stay public-safe.

## Do Not Report Private Data Publicly

If you believe private mailbox content, tokens, credentials, claim secrets, banking data, or private local paths were accidentally published, do not open a public issue with the secret value.

Email the repository owner instead:

taher.dhanera@gmail.com

## Scope

In scope:

- accidental disclosure in this public repository;
- unsafe demo data;
- links that expose private material;
- scripts or docs that could encourage spam or unsafe automation.

Out of scope:

- third-party bounty platforms;
- unrelated GitHub repositories;
- speculative reports without reproducible evidence.
