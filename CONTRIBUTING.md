# Contributing

This repository is a public-safe demo of an agentic operations workflow.

Good contributions:

- improve the public dashboard readability;
- add clearer sanitized examples;
- tighten the no-spam decision policy;
- improve docs for GitHub/CI/claim-state workflows;
- add tests or validators for sanitized demo data.

Do not contribute:

- private mailbox content;
- authenticated claim URLs;
- cookies, tokens, credentials, or bank/payout details;
- copied private workspace paths;
- comments or scripts that encourage spammy bounty claiming.

Keep contributions small, reviewable, and evidence-backed.
